# Keil快速创建工程

# 背景说明

第一次使用keil软件的时候,我们都安装过pack包,pack包中不只是提供了芯片的烧写地址这些内容更包括了芯片对应的启动文件,以及所有的固件库文件.使用Manage Run-Time Environment可以帮助我们快速创建不同芯片对应的工程文件(国产芯片也支持).
